# Openvino API Examples with Open Model

## Requirement
```
pip install numpy==1.26.4 opencv-python openvino scikit-learn scipy
```

## Get started
```
git clone https://github.com/Jaemnani/openvino_api.git
cd openvino_api
python main.py
```
